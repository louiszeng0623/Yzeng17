## 📱 如何订阅到日历（强烈推荐 iCloud）

1. 打开下面的链接：
   ```
   https://louiszeng0623.github.io/Yzeng17/calendar.ics
   ```
2. 点击 **订阅**
3. 选择 **iCloud 同步**
4. 完成 ✅  
   （以后比赛更新会自动推送到日历）

## 🔄 自动更新机制

| 内容 | 说明 |
|---|---|
| 数据源 | 懂球帝（球队页面赛程） |
| 更新频率 | 每 6 小时自动爬取 |
| 技术栈 | Python + Playwright + GitHub Actions |
| 输出格式 | 标准 `.ics`，支持所有日历系统 |
